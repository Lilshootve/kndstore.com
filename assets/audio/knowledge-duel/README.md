# Knowledge Duel Audio Assets

Place the following audio files in this directory:

- `bgm-arena.mp3` - Ambient loop during battle
- `sfx-correct.mp3` - Correct answer
- `sfx-wrong.mp3` - Wrong answer
- `sfx-hit-enemy.mp3` - Damage to enemy
- `sfx-hit-player.mp3` - Damage to player
- `sfx-victory.mp3` - Battle victory
- `sfx-defeat.mp3` - Battle defeat
- `sfx-level-up.mp3` - Avatar level up

The KnowledgeDuelAudio manager works without these files (silent fallback); add them for full experience.
