# KND Drop Chamber Audio Assets

Place the following audio files in this directory for the premium drop experience:

- `sfx-scan.mp3` - Soft loop during capsule scan
- `sfx-reveal-common.mp3` - Gentle click for common
- `sfx-reveal-rare.mp3` - Brighter chime for rare/special
- `sfx-reveal-epic.mp3` - Short fanfare for epic
- `sfx-reveal-legendary.mp3` - Epic fanfare for legendary

The DropAudioManager works without these files (silent fallback); add them for full experience.
