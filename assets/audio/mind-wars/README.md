# Mind Wars Audio Assets

Place the following audio files in this directory:

- `bgm-arena.mp3` - Ambient loop for battle
- `sfx-attack.mp3`, `sfx-defend.mp3`, `sfx-ability.mp3`, `sfx-special.mp3`
- `sfx-hit.mp3`, `sfx-crit.mp3`, `sfx-heal.mp3`, `sfx-evade.mp3`
- `sfx-victory.mp3`, `sfx-defeat.mp3`, `sfx-turn.mp3`

The AudioManager will work without these files (silent fallback); add them for full experience.
